FIELD · PLAYGROUND

f(x, y, t) = x,y ∈ [0,1] · t = 秒 · 返り値 = 表示値 0..1（小=暗 / 大=明、外側はクランプ） · 先頭に f(x,y,t)= も可
MANUAL — 関数・定数リファレンス

VARIABLES — 式の中で使える値

x, y
評価中のセルの位置。画面を [0,1]×[0,1] に正規化した座標。左上=(0,0)、右下=(1,1)。
t
経過時間（秒）。毎フレーム増える。式に入れると絵が動く。
seed
乱数の種（下の入力欄で変える）。noise/rnd に自動で効くので、値を変えると模様が振り直る（手で混ぜる必要なし）。

CONSTANTS — 定数

PI
円周率 π ≈ 3.14159。角度はラジアン（半周 = PI）。
TAU
2π ≈ 6.283 = 1周分の角度。sin(x*TAU) は x が 0→1 で正弦波ちょうど1周。

OPERATORS — 演算子

+  -  *  /
加算・減算・乗算・除算。
%
剰余（余り）。mod() と違い負の結果もあり得る。
^
べき乗（a^b = a の b 乗）。右結合。
( )
グループ化。

優先順位: ^ > 単項マイナス > * / % > + -　（例: -2^2 = -4）

COMMENTS — コメント（評価されないメモ）

// …
行コメント。// から行末まで無視。例: sin(x*8) // 横縞
/* … */
ブロックコメント。複数行いける。例: a /* ここは無視 */ + b
Ctrl + /
行コメントのトグル（Mac は ⌘ + /）。カーソル行、選択時は触れた行すべて。VSCode と同じ。

FUNCTIONS · 三角 — 角度はラジアン（座標の数学に）

sin(a) cos(a)
正弦・余弦。返り値 [-1,1]。明暗の縞にするときは sin(x*8)*0.5+0.5 のように 0..1 へ写す（シェーダ流のイディオム）。
atan2(y, x)
点 (x,y) の方向角 [-π,π]。中心からの角度＝放射・渦に。

FUNCTIONS · 数値

abs(x)
絶対値（符号を消す）。リング abs(dist(…)-r) などに。
floor(x)
切り捨て（x 以下の最大の整数）。
fract(x)
小数部 = x − floor(x)。[0,1) を繰り返す（鋸波）。タイル状の模様に。
sqrt(x)
平方根（x<0 は 0 扱い）。
mod(a, b)
剰余。常に [0, b) に収まる（負でも正に巻く）。繰り返し・タイル化に。

FUNCTIONS · 混合・しきい値

min(…) max(…)
引数の最小 / 最大。形の合成に。
clamp(v, a, b)
v を範囲 [a, b] に収める（はみ出しを切り落とす）。
mix(a, b, t)
a と b を t で線形補間。t=0→a、t=1→b、0.5→中間。
step(edge, x)
固いしきい値。x < edge なら 0、そうでなければ 1（2値化）。
smoothstep(e0, e1, x)
なめらかなしきい値。x が e0→e1 を渡る間に 0→1 へ S字で上がる。外側はクランプ。境界をぼかす・グラデに。

FUNCTIONS · 場（パターン生成）

dist(x0,y0, x1,y1)
2点 (x0,y0)–(x1,y1) の距離。円・放射状の模様に。
noise(x, y)
なめらかな値ノイズ。近い座標は近い値 → 雲・有機的なムラ。[0,1]。引数の倍率で細かさを変える（例 noise(x*8, y*8)）。seed で模様が変わる。
fbm(x, y[, oct])
noise を倍率を変えて oct 回重ねたフラクタルノイズ（既定 oct=4）。雲・地形のディテール。[0,1]。
rnd(x, y)
座標から決まる無状態の擬似乱数 [0,1]。隣どうしは不連続（ザラザラ）。seed で振り直る。

VALUE BLOCK — 文を並べて最後に値を返す

式 1 本のほか、代入と repeat を並べてから最後に値の式を 1 つ置ける（julia の脱出時間や N 個の総和など）。文は改行か ; で区切る。

name = 式
変数の代入。s = 0 など。
repeat N { … }
N 回繰り返す。repeat N as i { … } で番号 i（0..N-1）が使える。総和や反復に。
最後の式
ブロックの値（＝場の値 0..1）。1 つだけ置く。

例: 上の「★ julia」ボタン（脱出時間を repeat で反復）。
